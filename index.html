    Methodology

    Approved sources are searched first: DLD, Dubai REST, Dubai Pulse (government), Gulf News, Khaleej Times, The National (press), Bayut, Property Finder, DXB Interact (portals), Propsearch, fäm Properties, Global Property Guide (research).

    Source ranking: Government data is treated as highest authority. Press and research sources are used for context and trend analysis.

    Fallback: If approved sources have insufficient data, the best available public source is used and flagged as unverified.

    Freshness: All data points include the period or date they refer to. The agent does not present historical data as current.

    Confidence: Findings cross-verified across multiple sources are treated as confirmed. Single-source findings are presented as reported.

    Methodology & Sources

    Approved sources are searched first and prioritised:

    🟢 Government (Tier 1): Dubai Land Department, Dubai REST, Dubai Pulse

    🟡 Press (Tier 2): Gulf News Property, Khaleej Times, The National

    🔵 Portals (Tier 3): Bayut, Property Finder, DXB Interact

    🟣 Research (Tier 4): Propsearch.ae, fäm Properties, Global Property Guide

    Fallback: If no approved source has sufficient data, the best available public source is used and flagged as unverified.

    Anti-hallucination: The agent never invents numbers. Any figure that cannot be sourced is omitted entirely. Every metric includes its source and time period.

    Confidence levels: Cross-verified findings (2+ sources) are treated as confirmed. Single-source findings are presented as reported. Inferred data is explicitly labelled.
